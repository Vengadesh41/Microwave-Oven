# Microwave Oven Simulation

## Overview

The Microwave Oven Simulation is an Embedded C project developed using the PIC16F877A microcontroller. The project simulates the basic functionality of a real microwave oven through a menu-driven interface. Users can select different cooking modes, set cooking time, and monitor the cooking process using an LCD display.

The objective of this project is to understand real-time embedded system design, peripheral interfacing, timer management, and state machine implementation.

---

## Features

* Multiple cooking modes

  * Micro Mode
  * Grill Mode
  * Convection Mode
  * Start Mode
* User-friendly menu-driven interface
* Cooking time configuration
* Real-time countdown timer
* LCD display for status updates
* Matrix keypad for user input
* Buzzer notification after cooking completion
* Pause, Resume, and Stop functionality
* Safety checks during operation

## Project Workflow

1. Initialize all peripherals.
2. Display the main menu on the LCD.
3. Read user input from the keypad.
4. Select the desired cooking mode.
5. Set the cooking time.
6. Start the countdown timer.
7. Continuously update the LCD with the remaining time.
8. Allow Pause, Resume, or Stop operations.
9. Activate the buzzer when cooking is completed.
